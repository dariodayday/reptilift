REPTILIFT — exercise cartoon images
====================================

Drop a PNG here named  <exercise-id>.png  and it shows up automatically
next to that exercise in the picker and in the workout cards.
If an image is missing, the app falls back to the icon badge — so you
can add them one at a time. Square-ish images on a transparent (or dark)
background look best.

Tip: generate a consistent cartoon set (e.g. ChatGPT) — "simple flat
cartoon of a person doing a <exercise>, green on dark, square."
